# 🎬 FreeFlixx — Entertainment Menu System

A Python-based console application simulating a streaming platform menu, built as a beginner Python project. Users can browse movies, rent web series, and select OTT subscriptions — with a running price tracker.

---

## 📌 Features

- **Movies Menu** — Browse and add popular Bollywood films to your watchlist
- **Rental Movies** — Select paid web series (e.g. Mirzapur, Sacred Games) with individual pricing
- **Subscription Plans** — Choose from OTT platforms like Netflix, Hotstar, Amazon Prime, and more
- **Price Tracker** — Automatically calculates total cost of selected rentals and subscriptions

---

## 🛠️ Tech Stack

- **Language:** Python 3
- **Environment:** Jupyter Notebook / IDLE / Any Python IDE

---

## 🚧 Known Limitations

- No actual login/authentication logic implemented yet
- `break` outside loop error occurs when selecting `0` at the main menu (fix in progress)
- No persistent data storage — selections reset on each run
